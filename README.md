# 電商網站 - 流亡購物網
> 專案作品為虛擬購物商店，以線上遊戲'流亡黯道'為主題製作。

### 簡介

前台功能：

* 商品展示
* 購物車及模擬簡易結帳

後台功能：

* 產品管理
* 訂單管理
* 會員管理

### DEMO

[流亡購物網](https://skyery.github.io/Skyery-Practice-ShoppingCart/)

### 專案使用

* Vue3.js
* Vue CLI
* Vuex
* Vue Router
* JavaScript (ES6、LocalStorage...)
* FireBase (Realtime Database、Authentication...)
